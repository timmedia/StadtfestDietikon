Funktionsweise

1. Mit Hil­fe der Kar­te auf dies­er Web­sei­te su­chen Sie ein­en be­lie­bi­gen Pos­ten.

2. Nach­dem Sie das Schild am Pos­ten ge­fun­den ha­ben, ge­ben Sie den Code, be­steh­end aus ver­schie­den­en For­men, auf dies­er Sei­te ein.

3. Ist der Code rich­tig, so wird ein Vi­deo an­ge­zeigt.

Schulprojekt

Dies ist ein Pro­jekt der Schu­le Die­ti­kon für das Stadt­fest vom Au­gust 2018, um­ge­setzt in Zu­sam­men­ar­beit mit City-OL.

Positionsanzeige auf Apple-Geräten

Falls Sie ein mo­bi­les Ge­rät der Mar­ke Ap­ple nut­zen, kann stan­dard­ge­mäss Ihre Pos­i­tion auf der Kar­te nicht an­ge­zeigt wer­den. Den Ort­ungs­dients kön­nen Sie un­ter Ein­stel­lung­en > Da­ten­schutz > Or­tungs­dien­ste > Sa­fari frei­schal­ten. Mehr In­for­ma­tio­nen hier.
